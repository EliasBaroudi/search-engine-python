Detailed Description

Script principal de l'application.

Ce fichier permet la création de nos objets CVE ainsi que de notre corpus. Dans un premier temps, nous recherchons les CVE les plus connues à l'aide de l'API Kevin. Ensuite, nous créons le corpus à partir de ces CVE. Enfin, nous sollicitons le moteur de recherche avec une requête contenant des mots clés.
